# Optical Tweezers Analysis
This project focuses on the simulation and analysis of particle motion in optical tweezers.

## Objectives
- simulate Brownian motion
- Track particle trajectories
- Estimate trap stiffness
- Analyze particle motion using Mean Square Displacement (MSD)

## Tools
- Python
- NumPy
- Matplotlib
- SciPy

This repository is part of my research portfolio in optical physics and data analysis.
